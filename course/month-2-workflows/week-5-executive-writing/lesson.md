# Week 5: Executive-Ready Writing — Memos, Briefs, Proposals, Summaries

## The Executive Communication Mindset

Executives need: speed, clarity, decisions. Every document should answer three questions:
1. What is this about?
2. Why should I care?
3. What do you want me to do?

## Prompt Patterns for Professional Writing

### Memo Pattern
```
Role: You are a [job title] writing to [audience].
Task: Write a one-page memo about [topic].
Structure:
- Purpose (2 sentences)
- Background (3-4 sentences)
- Key findings/recommendations (3 bullet points)
- Next steps (2-3 bullets)
Tone: Direct, professional, no jargon.
Constraint: Max 300 words.
```

### Briefing Document Pattern
```
Role: You are preparing a briefing for [person] ahead of [meeting/event].
Task: Summarize the key information needed.
Structure:
- Situation overview
- Key stakeholders
- Positions/risks
- Recommended talking points
Constraint: Bullet format, no paragraphs longer than 2 sentences.
```

### Proposal Pattern
```
Role: You are pitching [idea] to [decision-maker].
Task: Write a proposal outline.
Framework: Problem → Solution → Evidence → Cost/Effort → Ask
Tone: Persuasive but factual.
Constraint: Include one paragraph of counterargument and your response.
```

## One-Page Discipline
Always include a length constraint. If output exceeds one page, iterate:
"Shorten to 5 bullet points. Remove all background context."
"Rewrite this as a single paragraph of no more than 100 words."

## Common Writing Flaws AI Can Fix
- Passive voice → active voice
- Jargon → plain language
- Long sentences → short, scannable chunks
- No structure → clear hierarchy of ideas
