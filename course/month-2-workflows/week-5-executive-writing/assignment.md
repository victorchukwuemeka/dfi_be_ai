# Assignment: Professional Writing Portfolio

## Task
Create 3 professional documents using AI, then refine each.

## Requirements
1. Choose 3 scenarios from your real work (or realistic simulations):
   - A memo or status update
   - A proposal or recommendation
   - A difficult email (decline, feedback, bad news)
2. For each:
   - Write a structured prompt using the patterns from this week
   - Generate the first draft
   - Refine with at least 2 iteration rounds (cite what you changed and why)
   - Show before/after

## Deliverable
A document with all 3 scenarios, each showing:
- Scenario context (who, what, why)
- The structured prompt
- First output
- Refinement 1 (what changed, why)
- Refinement 2 (what changed, why)
- Final output

## Grading Criteria
- 3 different document types
- Clear prompt structure for each
- At least 2 visible refinements per document
- Refinements are strategic (targeting specific weaknesses)
- Final outputs are professional and usable
