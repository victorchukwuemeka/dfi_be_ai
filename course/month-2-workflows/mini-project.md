# Mini-Project: Business Communication Toolkit

## Due
End of Month 2

## Objective
Build a reusable toolkit of prompts, templates, and workflows for professional communication.

## Requirements

### Template Library (at least 6 templates)
Create prompt templates for:

1. **Meeting follow-up** — notes → action items → email summary
2. **Status update** — project update for stakeholders
3. **Decision memo** — options → analysis → recommendation
4. **Difficult message** — decline, feedback, or bad news
5. **Briefing document** — background → key points → talking points
6. **Synthesis report** — multiple sources → patterns → recommendation

Each template must be:
- A reusable prompt with `[placeholders]`
- Tested with real or realistic content
- Stored in Git with at least one refinement commit

### Workflow Documentation
For each template, document:
- When to use it
- Input needed
- Output format
- Typical refinement steps

### Demonstration
Pick 3 templates and demonstrate them with real content. Show:
- The filled-in prompt
- The AI output
- At least one refinement
- The final version
