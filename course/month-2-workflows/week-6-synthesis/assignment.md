# Assignment: Real Synthesis Project

## Task
Take a real situation where you have multiple sources of information and produce a synthesized output with a recommendation.

## Requirements
1. Minimum 3 sources (meeting notes, emails, reports, articles, feedback)
2. Produce:
   - A synthesis document (findings, patterns, gaps)
   - A recommendation with reasoning
   - A one-paragraph executive summary
3. Document your prompt and refinement process

## Deliverable
A document containing:
- Context: what was the situation?
- All sources (redacted if needed)
- Initial synthesis prompt and output
- Refined prompt and output
- Final recommendation
- Reflection: what did AI synthesis add vs doing it manually?

## Grading Criteria
- 3+ sources used
- Synthesis identifies patterns, not just summaries
- Recommendation is specific and justified
- Refinement process is documented and shows improvement
- Honest reflection on AI's added value
