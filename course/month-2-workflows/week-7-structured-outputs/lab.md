# Lab: Build Structured Outputs from Unstructured Input

## Objective
Convert unstructured information into tables, rankings, and action plans.

## Instructions

### Part 1: Unstructured Input
Find or write 3-4 paragraphs of unstructured text about a decision or project.

### Part 2: Three Structured Formats
Use AI to convert that text into:

1. **Comparison table** — if input compares options
2. **Scored ranking** — rank the options with criteria
3. **Action plan table** — with owners and deadlines

### Part 3: Refine
1. Pick the output that needs most improvement
2. Add a constraint: "Add a rationale column" or "Sort by priority"
3. Add a second constraint: "Include status indicators"
4. Compare original vs refined

## Deliverables
- Unstructured input text
- All 3 structured outputs
- Before/after of the refined output
- Which output format was most useful for your content?
