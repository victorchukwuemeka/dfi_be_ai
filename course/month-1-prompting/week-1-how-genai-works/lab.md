# Lab: Temperature and Prompt Structure Comparison

## Objective
Experiment with temperature settings and prompt structure to see how they affect output.

## Instructions

### Part 1: Temperature Test
1. Pick a simple prompt: "Name 5 ideas for team building activities"
2. Run it with temperature 0 (or lowest setting)
3. Run it with temperature 1.5 (or highest setting)
4. Compare: How does creativity/randomness change?

### Part 2: Vague vs Structured
Run these two prompts and compare results:

**Vague:**
"Explain machine learning."

**Structured:**
"You are a tech journalist writing for business executives who are not technical. Explain machine learning in 4 sentences. Use an analogy in the first sentence. Avoid jargon."

### Part 3: Context Window
1. Copy a 200-word article
2. Prompt: "Summarize this in 2 sentences"
3. Now add the full article as context and prompt: "Summarize this in 2 sentences, focusing only on financial implications"

## Deliverables
- Paste both outputs for each part
- Write 1 sentence per part: what did you learn?
