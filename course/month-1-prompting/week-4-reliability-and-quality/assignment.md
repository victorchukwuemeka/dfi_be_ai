# Assignment: Iterative Prompt Improvement

## Task
Take one prompt through 3 rounds of evaluation and refinement.

## Requirements
1. Start with a prompt for a real task
2. Round 1: Run, score, identify gap
3. Round 2: Refine, run, score
4. Round 3: Refine again, run, score
5. Each round targets a different rubric dimension

## Deliverable
A document showing:
- Starting prompt and output
- Round 1: score, gap identified, refinement made, new output
- Round 2: score, gap identified, refinement made, new output
- Round 3: score, gap identified, refinement made, new output
- Final scores compared (table or chart)

## Grading Criteria
- 3 clear refinement rounds documented
- Each round targets a specific rubric dimension
- Scores improve (or honest explanation if they don't)
- Refinements are specific (not "make it better")
- Final output is measurably better than the first
