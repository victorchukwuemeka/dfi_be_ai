# Lab: Evaluate and Improve

## Objective
Practice the evaluate-refine cycle using a quality rubric.

## Instructions

### Part 1: Evaluate
1. Take a prompt you wrote earlier in this course
2. Run it and get the output
3. Score the output using the quality rubric (accuracy, clarity, tone, usefulness, completeness)
4. Identify the weakest dimension

### Part 2: Refine
1. Add a constraint targeting the weakest dimension
2. Re-run with the exact same input
3. Score again — did the weak dimension improve?

### Part 3: Self-Check
Add to your prompt: "After your response, list any assumptions you made and any facts you're unsure about."

Does the model identify issues you missed?

## Deliverables
- Initial prompt + output + scores
- Refined prompt + output + scores
- Comparison showing improvement
- Reflection: what did the self-check reveal?
