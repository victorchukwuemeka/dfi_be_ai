# Assignment: Advanced Prompting Showcase

## Task
Choose a real task (or create a realistic scenario) and demonstrate all three advanced techniques.

## Requirements
1. Pick a task that benefits from structure (e.g., data extraction, analysis, classification, drafting)
2. Create:
   - A zero-shot baseline prompt
   - A few-shot prompt with 3-5 examples
   - A chain-of-thought variant
   - An output-schema-enforced variant
3. Run all four on the same input
4. Document the outputs

## Deliverable
A document with:
- The task description
- All 4 prompts
- All 4 outputs
- A comparison table (accuracy, consistency, usability)
- Conclusion: which approach worked best and why

## Grading Criteria
- All 4 variants submitted
- Input is identical across all variants (fair comparison)
- Comparison table is specific and honest
- Conclusion shows understanding of when to use each technique
