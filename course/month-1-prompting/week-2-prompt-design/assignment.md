# Assignment: Prompt Design Portfolio

## Task
Create 5 structured prompts for real tasks you face in your work (or simulated work tasks).

## Requirements
Each prompt must include at least 3 of the 5 components:
- Role
- Task (specific, measurable)
- Constraints (length, tone, format)
- Examples (at least 1)
- Output schema

## Deliverable
A document with:
1. Each prompt written out
2. The output from running the prompt
3. A brief note on what you would refine next time

## Example Topics
- Summarize a long email thread
- Draft a project status update
- Brainstorm 10 blog post titles
- Analyze survey responses
- Write a difficult email (feedback, decline, etc.)

## Grading Criteria
- 5 prompts submitted
- Each prompt has clear components labeled
- Output is included for each
- At least one refinement note per prompt
