# Capstone: End-to-End AI Workflow

## Due
End of Month 3 (final submission)

## Objective
Build and document one complete AI-assisted workflow that solves a real problem, with measurable before/after impact.

## Full Requirements

### 1. Problem Statement
- What problem are you solving?
- Who faces this problem?
- What is the current cost (time, quality, effort, frustration)?
- Why is this problem worth solving?

### 2. Before State
Document how this task was done before using AI:
- Time required
- Steps involved
- Quality level (with evidence if possible)
- Pain points

### 3. Workflow Documentation
Full step-by-step documentation:

**For each step:**
- What is the input?
- What is the prompt? (exact text with [placeholders])
- What is the expected output?
- What quality check do you apply?
- What refinement did you make?

**Must include:**
- At least 4 distinct AI interactions
- At least 2 refinement iterations on at least 2 steps
- Clear human touchpoints (where you make judgments)

### 4. Evidence
- Screenshots or pasted outputs at each stage
- Git log showing version history
- Before/after comparison

### 5. After State
- Time required with the new workflow
- Quality level (compare to before)
- What improved? Be specific.
- What didn't improve? Be honest.

### 6. Reflection (500+ words)
- What was the most valuable lesson from this process?
- Which prompt technique made the biggest difference?
- Where did AI fall short? How did you compensate?
- What would you do differently next time?
- How will you apply this approach going forward?

### 7. Repository
All capstone work must be in a `capstone/` directory with:
- `proposal.md` — the approved proposal
- `workflow.md` — full workflow documentation
- `evidence/` — outputs, screenshots
- `reflection.md` — final reflection
- Git history showing the evolution of your work

## Submission
1. Repository link (GitHub/GitLab) or zipped repo with `.git`
2. `git log --oneline --graph --all`
3. A 2-minute video or written executive summary (200 words)

## Grading Criteria

| Criteria | Weight |
|----------|--------|
| Problem is real and well-defined | 10% |
| Before state is clearly documented | 10% |
| Workflow is complete (4+ AI interactions) | 20% |
| Refinement is documented and shows improvement | 15% |
| After state shows measurable impact | 15% |
| Reflection is thoughtful and specific | 15% |
| Repository is well-organized with version history | 10% |
| Ethical considerations addressed | 5% |

## Ethical Considerations
In your reflection, address:
- What decisions did you delegate to AI? Should you have?
- Did the AI produce anything misleading? How did you catch it?
- What would happen if someone used this workflow without human review?
- Privacy: did you expose any sensitive data?
