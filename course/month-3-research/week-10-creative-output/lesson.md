# Week 10: Creative Output with Constraints — Messaging, Campaigns, and Ideation

## Creativity Within Constraints

Paradoxically, constraints improve creative output. A blank page is paralyzing. A specific brief is freeing.

## Ideation Patterns

### Brainstorming with Structure
```
Role: Creative strategist.
Task: Generate 10 ideas for [topic].
Constraints:
- Each idea must be 1 sentence
- Ideas should vary widely (not variations on one theme)
- Include at least 2 ideas that seem unconventional

After generating, rank by: novelty, feasibility, impact.
```

### Messaging Framework
```
Develop messaging for [product/initiative].
Target audience: [description]
Tone: [brand voice]

Structure for each message:
- Headline (max 8 words)
- Supporting sentence
- Call to action (max 5 words)

Provide 3 options focused on different angles:
1. Problem-focused
2. Benefit-focused
3. Emotion-focused
```

### Campaign Concepts
```
Create 3 campaign concepts for [goal].
Each concept should include:
- Core idea (1 sentence)
- Tagline
- Channel strategy (which platforms, why)
- Success metric

Constraints:
- Budget: [low/medium/high]
- Timeline: [weeks/months]
- Brand guidelines: [existing constraints]
```

## Creative Refinement

### Combining Ideas
```
Take these 3 ideas and combine the strongest elements of each into one concept:
Idea A: [summary]
Idea B: [summary]
Idea C: [summary]
```

### Iterative Expansion
Start broad, then narrow:

Round 1: "Generate 20 ideas for [topic]"
Round 2: "Cluster these into themes and pick the strongest theme"
Round 3: "Develop that theme into 3 detailed concepts"
Round 4: "Flesh out the best concept with execution details"

## Breaking Writer's Block
Prompt: "I need to write about [topic] but I'm stuck. Ask me 5 questions that would help you write a first draft. I'll answer, then you write it."
