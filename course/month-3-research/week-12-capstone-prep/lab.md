# Lab: Capstone Proposal Development

## Objective
Draft and refine your capstone proposal.

## Instructions

### Part 1: Brainstorm
1. List 3-5 problems you could solve with an AI workflow
2. For each, write 2-3 sentences describing the workflow
3. Use AI to evaluate which has the strongest before/after impact

### Part 2: Draft Proposal
Use this prompt template:
```
I'm planning a capstone project on [topic]. Draft a one-page proposal:
1. Problem: what's the current pain point?
2. Proposed workflow: 4-6 steps
3. Expected impact: what would improve?
4. Success criteria: how will I measure success?

Target audience: [instructor/manager].
Be specific and practical.
```

### Part 3: Refine
1. Review the draft — what's missing?
2. Add more detail to the workflow steps
3. Strengthen the before/after comparison
4. Make success criteria measurable

### Part 4: Version Control
1. Create a `capstone/` directory in your course repo
2. Save the proposal as `proposal.md`
3. Commit the draft
4. Apply refinements and commit again

## Deliverables
- Your capstone proposal (final version)
- Git log showing the proposal evolution
- A note: what did AI add to your proposal that you wouldn't have included?
