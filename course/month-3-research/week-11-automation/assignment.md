# Assignment: SOP Library

## Task
Create 3 Standard Operating Procedures for AI-assisted workflows you use (or would use) in your work.

## Requirements
Each SOP must include:
- When to use it
- Input needed
- At least 3 steps with exact prompts (using [placeholders])
- Quality check after each step
- Error handling (what to do if output is poor)

Topics (choose 3):
- Content creation and editing
- Research and synthesis
- Email or communication triage
- Data analysis and reporting
- Meeting preparation and follow-up
- Decision support
- Project planning

## Deliverable
A document (or repository) with 3 complete SOPs, each tested and refined:
- Initial version
- Test results (run with real input)
- Refined version

## Grading Criteria
- 3 SOPs with clear structure (30%)
- Each has 3+ steps with exact prompts (25%)
- Quality checks defined for each step (15%)
- Evidence of testing with real input (15%)
- Refinements based on test results (15%)
