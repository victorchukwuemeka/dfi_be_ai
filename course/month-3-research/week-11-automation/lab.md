# Lab: Build a Prompt Chain

## Objective
Create and test a reusable prompt chain for a multi-step workflow.

## Instructions

### Part 1: Identify a Workflow
Choose a common task that requires 3-5 steps. Examples:
- Meeting notes → action items → email summary
- Raw data → analysis → recommendation
- Topic → outline → draft → polish

### Part 2: Design the Chain
Write out each step:
- What is the input?
- What is the prompt?
- What is the expected output?
- What is the quality check?

### Part 3: Test the Chain
1. Run through all steps with real input
2. Did any step produce poor output? Refine that step's prompt
3. Run again with different input

### Part 4: Document as SOP
Write the chain as an SOP using the template from the lesson.

## Deliverables
- The chain design (all steps with prompts)
- Output from testing (show each step's result)
- The refined chain (after testing)
- The SOP document
- Reflection: where did the chain break or need human judgment?
