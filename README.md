# Generative AI Course Outline (3 Months) — Professional Tool User Track

## Course Overview
A concise, practical GenAI course for professionals who want to use AI tools effectively in real work. The focus is high-signal workflows, strong prompting, verification habits, and measurable outcomes.

## Target Audience
- Professionals across roles who want to work faster and smarter with GenAI tools
- No coding required

## Learning Outcomes
By the end of the course, learners will be able to:
- Track and manage work with version control (Git)
- Design prompts that deliver consistent, high-quality results
- Build repeatable workflows for writing, research, analysis, and planning
- Evaluate and improve AI outputs with clear standards
- Apply safety, privacy, and compliance basics in day-to-day use
- Deliver a capstone workflow with documented impact

## Course Structure
- Duration: 3 months
- Format: Weekly lesson + lab + assignment
- Prerequisite week: Version Control (Git)
- Milestones: Monthly mini-projects + final capstone

---

## Prerequisite Week: Version Control (Git)
- Why version control matters for AI-assisted work
- Git basics: init, add, commit, status, log, diff
- Branching and merging fundamentals
- Working with remote repositories (GitHub/GitLab)
- Best practices: commit messages, .gitignore, README conventions
- Using Git alongside AI tools for iterative improvement

## Month 1: Prompting, Quality, and Reliability
- How GenAI works in plain terms: context, tokens, and sampling
- Prompt design: role + task + constraints + examples + output schema
- Advanced prompting: few-shot patterns and controlled outputs
- Reliability tactics: clarifying questions, validation steps, and self-checks
- Quality rubric: accuracy, clarity, tone, and usefulness
- Mini-project: Prompt library + quality checklist for a real workflow

## Month 2: Professional Workflows (Writing, Analysis, Planning)
- Executive-ready writing: memos, briefs, proposals, and summaries
- Synthesis and insight: turning messy notes into decisions
- Structured outputs: tables, comparisons, scoring, and action plans
- Collaboration: review loops, versioning, and human edits
- Mini-project: Business communication toolkit with templates

## Month 3: Research, Creativity, and Capstone
- Research workflow: scoping, question refinement, and verification
- Creative output with constraints: messaging, campaigns, and ideation
- Automation basics: reusable prompt chains and SOPs
- Capstone: Build one end-to-end workflow with before/after impact

---

## Assessment Breakdown
- Monthly mini-projects: 60%
- Participation and practice: 10%
- Capstone: 30%

## Required Resources
- Access to any GenAI tool (web or app)
- Git installed locally or access to GitHub/GitLab
- Google Docs or Word for assignments
